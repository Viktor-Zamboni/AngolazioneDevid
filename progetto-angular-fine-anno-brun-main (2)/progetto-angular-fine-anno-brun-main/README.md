## COME USARE L'APP?
NEL TERMINALE:
- cd LastProject
- npm i -g @angular/cli
- npm i
- ng serve
(Potrebbe essere richiesta l'installazione di tailwind css  e yarn, in tal caso: 
tailwind css: npm install -D tailwindcss 
yarn: npm install --global yarn
)
npm run ng serve -- --host=0.0.0.0 --disable-host-check
